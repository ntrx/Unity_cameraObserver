# Unity_cameraObserver
Rotation and zooming to centered gameobject

How to use:

1. create empty parent gameobject for mainCamera (pos. Vector3.zero, rot. Vector3.zero)
2. Attach cameraZoom.cs to mainCamera object
3. Use horizontal axis for horizontal view adjustment in Play
4. Use vertical axis for vertical view adjustment in Play
5. Using mouseScroll to adjust zoom in Play
